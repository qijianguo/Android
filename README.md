# 项目搭建(mvp)
###项目结构###
![](http://img.blog.csdn.net/20160612124751434)
>* bean—————————————————存放java model对象<br>
>* biz—————————————————–业务模块，根据不能业务建立子模块<br>
>* bridge————————————————-底层功能实现跟UI层的衔接层
>*  capabilities——————————————–底层功能具体实现（后期项目迭代到一定程度稳定后会考虑以jar形式导入）<br>
>* constant———————————————–常量<br>
>*  ui——————————————————界面，根据不同业务建立子模块<br>
>* util—————————————————–业务层公共方法<br>
>* view—————————————————自定义view实现<br>
